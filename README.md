
# Calma 

Calma is a privacy-focused iOS application designed to help users track, reflect, and understand their emotional wellbeing.
It combines structured mood logging, data visualisation, and optional AI-powered insights to encourage self-awareness and personal growth.



Technologies Used
- 
- Swift
- Groq API 

Attention !
- 
The project version uploaded on GitHub has no API key, in order to have working Insights, do the following:
- Go to [Groq](https://console.groq.com/home?utm_source=website&utm_medium=outbound_link&utm_campaign=dev_console_click)
- Create an account 
- Go to [API Keys](https://console.groq.com/keys)
- Create new API key
- Give a name and select your expiration period, and submit
- Copy the API key and keep it safe
- Go to Calma files at GroqService.Swift
- Paste your API key for 'apiKey' variable
- Run the application
- Congrats
